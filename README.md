# ✅ JavaScript Todo List

HTML, CSS, JavaScript만을 사용하여 구현한  
**투두리스트 웹 애플리케이션**입니다.  
프레임워크 없이 DOM 조작과 상태 관리를 직접 구현하는 것을 목표로 제작했습니다.

---

## 📌 프로젝트 개요

- **개발 목적**  
  순수 JavaScript를 활용한 DOM 조작, 이벤트 처리,  
  LocalStorage 기반 상태 관리 경험

- **개발 형태**  
  개인 프로젝트

- **진행 기간**  
  2025-01-11 ~ 2025-01-15(5일)

---

## 🛠 기술 스택

- **HTML5**
- **CSS3**
- **JavaScript**
- **LocalStorage**

---

## ✨ 주요 기능

### 1️⃣ 할 일 추가
- 입력값이 비어 있을 경우 추가 방지
- Enter 키 입력으로도 할 일 등록 가능

### 2️⃣ 할 일 완료 처리
- 체크박스를 통해 완료 상태 변경
- 완료 시 취소선 스타일 적용
- 완료된 항목에만 삭제 버튼 노출

### 3️⃣ 할 일 삭제
- 삭제 전 `confirm()`을 통한 사용자 확인

### 4️⃣ 필터 기능
- 전체 / 미완료 / 완료 상태별 필터링
- 동일 필터 버튼 재클릭 시 전체 보기로 복귀

### 5️⃣ 데이터 영속성
- LocalStorage를 활용하여
  페이지 새로고침 후에도 할 일 데이터 유지

## 🧠 구현 포인트

- **LocalStorage 기반 상태 관리**
  - todo 목록을 배열로 관리하고 JSON 형태로 저장
- **DOM 직접 조작**
  - `createElement`, `appendChild`를 활용한 동적 렌더링
- **UX 고려**
  - 완료 항목에만 삭제 버튼 표시
  - 필터 버튼 활성 상태 시각화
- **확장성**
  - 추후 React 기반 Todo 앱으로 확장 가능한 구조
